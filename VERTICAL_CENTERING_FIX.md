# Исправление вертикального центрирования элементов

## Проблема
В админ панели и профиле элементы в карточках были прижаты к верхней части, что выглядело неаккуратно. Элементы не были вертикально центрированы в своих рамках.

## Примеры проблемных мест (из скриншотов):
- Список товаров в админ панели
- Список категорий в админ панели
- Карточки профиля (Избранное, Мои заказы, Настройки)
- Панель администратора в профиле

## Решение

Добавлены **точечные** CSS правила в `src/index.css`, которые центрируют элементы вертикально только в конкретных местах:

### Основные изменения:

1. **Строки списков в админ панели**
   - Все строки в списках (товары, категории, заказы) теперь центрированы вертикально
   - Минимальная высота 60px для комфортного отображения

2. **Карточки профиля**
   - Карточки "Избранное", "Мои заказы", "Настройки" центрированы
   - Минимальная высота 80px
   - Иконки и текст выровнены по центру

3. **Flex контейнеры с иконками**
   - Все контейнеры с `items-start` изменены на `items-center` только внутри карточек
   - Иконки и текст в одной строке

## Технические детали

### Применяемые стили:
```css
/* Только для админ панели и профиля */
display: flex !important;
align-items: center !important;
min-height: 60px-80px !important;
```

### Охват:
- ✅ Админ панель (списки товаров, категорий)
- ✅ Профиль (карточки с иконками)
- ❌ НЕ затрагивает остальной дизайн
- ❌ НЕ меняет глобальные стили

## Результат

Теперь элементы в админ панели и профиле вертикально центрированы в своих контейнерах. Остальной дизайн остался без изменений.

### До:
- Элементы прижаты к верху в админ панели
- Элементы прижаты к верху в профиле
- Визуально неаккуратно

### После:
- Элементы центрированы вертикально в админ панели
- Элементы центрированы вертикально в профиле
- Остальной дизайн не изменен

## Совместимость

Изменения применяются точечно и не ломают существующую функциональность:
- Используется `!important` только для конкретных селекторов
- Затрагивает только админ панель и профиль
- Работает на всех устройствах и размерах экрана
- Не влияет на остальные страницы и компоненты
